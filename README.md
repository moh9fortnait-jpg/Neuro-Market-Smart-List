# 13. Neuro-Market Smart List

## Concept
A practical expense tracker designed for traditional weekly markets. It helps users manage their "Physical Wallet" by tracking spending in real-time and providing alerts before they exceed their set budget.

## Innovation
The project focuses on **Real-time Financial Discipline**. It uses a continuous loop logic to subtract expenses from a starting balance, providing immediate feedback on "Remaining Purchasing Power," which is crucial in fast-paced market environments.

## How to Run
1. Run: `python main.py`
2. Set your total market budget.
3. Add items and their prices as you shop.
4. Use 'finish' to see your total spending summary.
